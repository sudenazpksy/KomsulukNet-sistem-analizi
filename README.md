# KOMŞULUKnet — Komşuluk & Site Yönetim Platformu

Çok katlı konut sitelerinde aidat takibi, arıza bildirimi ve duyuru yönetimini dijitalleştiren rol tabanlı web platformunun sistem analizi ve tasarımı.

## Kapsam
- Paydaş analizi ve önceliklendirme matrisi
- MoSCoW yöntemiyle gereksinim analizi
- Use Case senaryoları
- Veritabanı tasarımı
- JWT kimlik doğrulama ve rol tabanlı erişim kontrolü
- Online ödeme entegrasyonu (Stripe/İyzico)

## Teknolojiler (Tasarım)
JavaScript · PHP · REST API · JWT
